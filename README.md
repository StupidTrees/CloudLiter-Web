# 云升Web

云升（CloudLiter）为轻量创意交友平台。“云升”代表着云上交友，轻便快捷、词云等轻量聊天APP的特性。
- 实际上代表着我们十分有限的学生机服务器资源（一升云）
- 实际上是给APP起名的那天晚上，董云升同学突然闯入给的灵感

## 仓库说明
本仓库为云升后端源代码，使用node.js+express框架搭建
